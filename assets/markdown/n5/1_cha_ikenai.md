# ちゃいけない・じゃいけない

**Arti:** Tidak boleh melakukan / dilarang (bahasa lisan / percakapan)

Tata bahasa ini adalah cara yang lebih santai (kasual) untuk mengatakan bahwa kita tidak boleh melakukan sesuatu. Untuk situasi yang lebih formal, kamu bisa menggunakan **てはいけない** (*te wa ikenai*).

### Cara Penggunaan

Gunakan kata kerja bentuk **て**, lalu hilangkan huruf **て**-nya.
* **Kata Kerja (Verb) て** ➔ ちゃだめ / ちゃいけない / ちゃいけません
  * *Contoh:* たべる ➔ たべて ➔ **たべちゃ** (*taberu* ➔ *tabete* ➔ *tabe cha*)

Jika bentuk **て** berakhiran dengan **で**, maka **ちゃ** berubah menjadi **じゃ**.
* **Kata Kerja (Verb) で** ➔ じゃダメ / じゃいけない / じゃいけません
  * *Contoh:* のむ ➔ のんで ➔ **のんじゃ** (*nomu* ➔ *nonde* ➔ *nonja*)

### 3 Pola Akhiran Utama

1. **ダメ** (*dame*) = standar, kasual/santai
2. **いけない** (*ikenai*) = standar, kasual/santai
3. **いけません** (*ikemasen*) = arti yang sama, tapi lebih sopan/formal
